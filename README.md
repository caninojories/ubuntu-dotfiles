<h1 align="center"> Ubuntu Dotfiles </h1>

```bash
# download unzip
$ apt install unzip

# curl the repository
$ curl -L -O https://github.com/caninojories/ubuntu-dotfiles/archive/master.zip

# unzip
$ unzip master.zip

# cd to directory
$ cd ubuntu-dotfiles-master
```

## Running our dotfiles

```bash
# zshrc
$ sh dot.sh

# homebrew and packages
$ sh dev.sh

# software applications
$ ./applications
```
